# FactorioWeb 项目框架文档

> **最后更新**: 2026-03-11
> **版本**: v2.2

---

## 一、项目概述

FactorioWeb 是一个 Factorio 游戏服务器 Web 管理面板，提供服务器控制、Mod 管理、玩家管理、配置编辑、实时监控等功能。

### 技术栈

| 层级 | 技术 |
|------|------|
| **后端** | PHP 7.4+ (自定义 MVC 架构) |
| **WebSocket** | Workerman |
| **前端** | HTML + JavaScript + Bootstrap 5 |
| **HTTP 客户端** | Axios |
| **终端模拟** | XTerm.js |
| **配置存储** | JSON 文件 |
| **进程管理** | Screen + RCON 协议 |

## 三、核心模块说明

### 3.1 控制器层

| 控制器 | 职责 | 主要方法 |
|--------|------|----------|
| AuthController | 用户认证 | login, logout, checkAuth, generateHash |
| ServerController | 服务器控制 | start, stop, save, console, rconTest, isRunning |
| ChatController | 聊天设置 | getSettings, saveSettings, addTriggerResponse |
| FileController | 文件管理 | list, upload, download, delete |
| SystemController | 系统监控 | stats, onlinePlayers, phpInfo |
| DaemonController | 守护进程 | start, stop, status, restart |
| LogController | 日志处理 | tail, stream |

### 3.2 服务层

| 服务 | 职责 |
|------|------|
| ServerService | 服务器启动/停止/状态检测/RCON通信 |
| RconService | Factorio RCON 协议实现 |
| AuthService | 用户认证/密码加密/Session管理 |
| ChatService | 聊天设置/自动响应配置 |
| FileService | 文件上传/下载/管理 |
| MonitorService | 系统资源监控 |

### 3.3 自动响应模块

功能包括：
- **周期性消息**: 定时发送服务器公告
- **关键词回复**: 自动回复玩家聊天
- **玩家事件**: 上线欢迎、下线告别
- **投票踢人**: 玩家投票踢出功能
- **物品请求**: 玩家可请求物品

## 八、自动响应系统

自动响应系统可以在服务端后台自动执行以下功能，无需保持浏览器打开。

### 8.1 功能概述

| 功能 | 说明 |
|------|------|
| 周期性消息 | 按设定时间间隔自动发送消息（15分钟/30分钟/每小时/每2小时/每6小时/每12小时/每天） |
| 定时任务 | 在指定时间点发送一次性消息（活动预告、维护通知等） |
| 关键词自动回复 | 监听游戏内聊天，当玩家发送包含特定关键词时自动回复 |
| 玩家事件响应 | 玩家加入/离开时自动发送欢迎/告别消息 |

### 8.2 支持的变量

| 变量 | 说明 |
|------|------|
| `{player}` | 发送消息的玩家名 |

### 8.3 安装步骤

**方法一：使用自动设置脚本（推荐）**

```bash
cd /www/wwwroot/factorio/web
./setupCron.sh
```

选择选项 1 安装 Cron 任务。

**方法二：手动设置**

```bash
crontab -e
# 添加以下行（每分钟执行一次）
* * * * * /usr/bin/php /www/wwwroot/factorio/web/autoResponder.php >> /www/wwwroot/factorio/web/autoResponder.log 2>&1
```

### 8.4 配置示例

**周期性消息**
- 消息内容：`欢迎游玩本服务器！请遵守游戏规则。`
- 间隔：每小时

**关键词回复**
- 关键词：`帮助`
- 响应内容：`{player}，请查看服务器公告或联系管理员获取帮助。`

**玩家事件**
- 上线欢迎：`欢迎 {player} 加入游戏！祝你游戏愉快！`
- 下线告别：`{player} 离开了游戏，再见！`

### 8.5 相关文件

| 文件 | 说明 |
|------|------|
| `autoResponder.php` | 自动响应系统主程序 |
| `autoResponderDaemon.php` | 守护进程模式 |
| `modules/autoResponder/` | 自动响应模块目录 |
| `config/chatSettings.json` | 配置文件存储 |
| `config/autoResponderState.json` | 运行状态记录 |
| `logs/autoResponderDaemon.log` | 运行日志 |

### 8.6 故障排除

**自动响应不工作**
```bash
# 检查 cron 任务
crontab -l

# 查看日志
tail /www/wwwroot/factorio/web/logs/autoResponderDaemon.log

# 手动测试
php /www/wwwroot/factorio/web/autoResponder.php
```

**关键词回复不触发**
1. 确保关键词匹配（支持部分匹配）
2. 检查日志中是否有聊天消息被正确解析
3. 确认 `factorio-current.log` 文件路径正确

**玩家事件不触发**
1. 确认玩家事件功能已启用
2. 检查日志中是否有 `[JOIN]` 和 `[LEAVE]` 标记
3. 确认消息内容不为空
