## 二、服务层函数详解

### 2.1 StateService (services/StateService.php)

| 方法 | 访问级别 | 参数 | 返回值 | 说明 |
|------|----------|------|--------|------|
| `__construct()` | public | $configDir | void | 构造函数 |
| `saveState()` | public | $name, $state | bool | 保存状态数据 |
| `loadState()` | public | $name | array | 加载状态数据 |
| `getStateValue()` | public | $name, $key, $default | mixed | 获取状态值 |
| `setStateValue()` | public | $name, $key, $value | bool | 设置状态值 |
| `deleteState()` | public | $name | bool | 删除状态 |

## 五、命名规范

### 5.1 命名规则

| 类型 | 规范 | 示例 |
|------|------|------|
| 函数名 | camelCase | getServerConfig() |
| 类名 | PascalCase | ServerService |
| 常量 | UPPER_SNAKE_CASE | MAX_RETRY_COUNT |
| 变量 | camelCase | $serverId |
| 私有属性 | camelCase | $stateService |

### 5.2 文件命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 服务类 | PascalCase + Service | StateService.php |
| 控制器 | PascalCase + Controller | AuthController.php |
| 配置文件 | camelCase + .json/php | chatSettings.json |
| 日志文件 | camelCase + .log | autoResponder.log |
