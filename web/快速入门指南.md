# Factorio Server Pro - 快速入门指南

## 🚀 新功能概览

### 1. 登录系统

**默认账号**:
- 用户名: `admin`
- 密码: `password`

**安全建议**: 首次登录后立即修改密码！

### 修改密码步骤:

1. 编辑 `web/config.php` 文件
2. 找到 `users` 数组
3. 使用 PHP 生成新的密码哈希:
```php
<?php
echo password_hash('你的新密码', PASSWORD_DEFAULT);
?>
```
4. 替换 `config.php` 中的密码哈希值

## 🛡️ 安全建议

1. **修改默认密码** - 首次使用立即更改
2. **使用 HTTPS** - 生产环境务必启用
3. **限制访问 IP** - 通过 Web 服务器配置

## 📞 故障排除

### 登录失败
- 检查 `config.php` 文件是否存在
- 确认用户名和密码正确
- 检查 PHP session 目录权限

### 页面样式异常
- 清除浏览器缓存
- 确认静态资源加载正常
- 检查浏览器控制台错误

### API 请求失败
- 检查 PHP 错误日志
- 确认文件权限正确
- 验证 Web 服务器配置
